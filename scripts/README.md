# Scripts

This directory is reserved for the numbered analysis scripts used to reproduce
the modelling workflow described in `PIPELINE.md`.

The expected run order is:

1. `01_preprocess.py`
2. `02_features.py`
3. `03_train.py`
4. `04_validate.py`
5. `05_shap.py`

Raw TMD rainfall observations are not redistributed in this repository.
